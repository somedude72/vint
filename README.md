## Overview

Welcome to the vint library!

Vint is a header only lightweight arbitrary precision integer and floating point library in c++. It provides a comprehensive and performant set of features, and it does not require any dependencies to compile other than the standard template library in c++. At least c++ 11 is required to compile vint. 

Head over to the [documentation page](https://somedude72.github.io/vint/home) for more information on the installation and usage of vint. Note that this project is a hobby project of mine, and as a result maintenance will not be guaranteed.
